# Landing Page - Vida em Ordem

Arquivos incluídos:
- index.html
- assets/vida-em-orddem-cover.png
- assets/vida-leve-logo.png, se disponível

Como usar:
1. Abra `index.html` no navegador para visualizar.
2. Procure por `SEU_LINK_DE_CHECKOUT_AQUI` no arquivo.
3. Substitua pelo link real do checkout da Kiwify, Hotmart ou outra plataforma.
4. Suba o `index.html` junto com a pasta `assets` no servidor.

Preço configurado na página:
R$19,90

Observação:
A página evita promessas de emagrecimento, cura ou resultado garantido, deixando a comunicação mais segura para anúncios.
